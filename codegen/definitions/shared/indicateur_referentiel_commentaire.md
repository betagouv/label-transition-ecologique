# Le commentaire sur un indicateur d'un référentiel

## Valeur texte
Représente un commentaire d'une epci apposé sur un indicateur du référentiel.
`epci_id/indicateur_id`

```yaml
IndicateurReferentielCommentaire:
  epci_id:
    type: String
  indicateur_id:
      type: String
  value:
      type: String
```
