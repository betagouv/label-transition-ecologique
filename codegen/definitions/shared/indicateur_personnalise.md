## Indicateur créé par une epci

Représente un indicateur créé par une epci.

Il est stocké à l'adresse `epci_id/uid`

```yaml
IndicateurPersonnalise:
    epci_id:
        type: String
    uid:
        type: String
    custom_id:
        type: String
    nom:
        type: String
    description:
        type: String
    unite:
        type: String
    meta: 
        type: Any
```
