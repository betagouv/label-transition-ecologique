# Valeur annuelle d'un indicateur personnalisé

## Valeur texte
Représente la valeur annuelle d'un indicateur personnalisé stockée à l'adresse:
`epci_id/indicateur_id/year`

```yaml
IndicateurPersonnaliseValue:
  epci_id:
    type: String
  indicateur_id:
      type: String
  year:
      type: num
  value:
      type: num
```
