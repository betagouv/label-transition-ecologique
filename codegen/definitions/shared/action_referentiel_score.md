## Action Référentiel Score

Représente le score d'une action du référentiel. Il est calculé à partir des status des actions.

```yaml
ActionReferentielScore:
    action_id:
        type: String
    action_nomenclature_id:
        type: String
    status:
        type: String
    points:
        type: num
    percentage:
        type: num
    potentiel:
        type: num
    referentiel_points:
        type: num
    referentiel_percentage:
        type: num
```
